📋 Template — Formularios HTML

1. Inputs básicos

/* Los tipos más comunes de <input> */

<!-- Atributos importantes -->
<input type="text"
       name="usuario"       /* clave que llega al server */
       id="usuario"         /* para enlazar con <label for> */
       placeholder="..."    /* texto de ayuda */
       required             /* campo obligatorio */
       disabled             /* deshabilitado */
       readonly             /* solo lectura */
       maxlength="50"       /* máximo de caracteres */
       value="default">

2. Select y Textarea

<select name="pais" multiple>
  <option value="ar">Argentina</option>
</select>

<textarea name="msg" rows="5" cols="40"></textarea>

3. Radio y Checkbox

/* Radio: solo 1 seleccionable por grupo (mismo name) */



/* Checkbox: múltiples seleccionables */

<!-- Radio: mismo name = mismo grupo -->
<input type="radio" name="grupo" value="op1">

<!-- Checkbox: name[] para enviar múltiples valores -->
<input type="checkbox" name="items[]" value="a">

4. Formulario completo con validación HTML5



/* Atributos de validación HTML5 */
required          → campo obligatorio
minlength / maxlength → largo del texto
min / max         → rango numérico
pattern="[A-Z]+"  → regex personalizada
type="email"      → valida formato email
type="url"        → valida formato URL
