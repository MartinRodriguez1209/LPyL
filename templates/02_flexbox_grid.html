🆚 Flexbox vs Grid — ¿Cuándo usar cada uno?

/* FLEXBOX → una dimensión (fila O columna) */
  - Navbar con items alineados
  - Botones en fila
  - Centrar un elemento
  - Lista de cards que pueden wrappear

/* GRID → dos dimensiones (filas Y columnas) */
  - Layout de página completa
  - Galería de imágenes
  - Formularios con múltiples columnas
  - Cualquier cosa con filas Y columnas al mismo tiempo

/* Truco: se pueden anidar. Grid para el layout general,
   Flex para los componentes internos */
